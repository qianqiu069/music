# 🎵 CloudMusicApp 仿网易云音乐移动端App
移动端开发课程设计 | 期末大作业
本项目仅用于课程学习与技术练习，严禁商用。

## 📖 项目简介
基于移动端开发技术复刻网易云音乐App核心交互与界面，完整实现本地音频播放、用户体系、歌单、搜索、评论等音乐App必备功能，满足移动端课程设计全部需求，包含完整业务逻辑、页面路由、本地存储与多媒体处理。

## 🛠 技术栈
原生安卓：Android Studio + Java/Kotlin + MediaPlayer + SQLite
Java调取音乐api接口
##功能
用户注册、登录、退出，个人资料修改、头像更换
首页推荐歌单、轮播图、热门榜单、新歌专区
全局悬浮音乐播放器：播放 / 暂停、切歌、进度拖拽、音量、循环 / 随机播放模式
逐行同步滚动歌词展示
全网歌曲搜索、歌手分类、歌曲详情
收藏单曲、自建歌单，支持增删歌曲
歌曲评论、点赞评论、查看热门评论
本地缓存播放记录、我的收藏
